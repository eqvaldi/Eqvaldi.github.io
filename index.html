    What is Eqvaldi Linux?¶

    Eqvaldi is a minimal, customized Linux distribution based on Debian and Armbian, targeting both x86 and ARM single-board computers. It ships a carefully selected package set to stay lean while still being usable out of the box.

    Key characteristics¶

    Feature Details
    Base Debian (Bookworm / Bullseye) + Armbian
    Desktops Cinnamon, MATE
    Architectures x86_64, ARM (various SBCs)
    Release lines V2-LTS · V3-LTS · V4-dev
    Kernel LTS + Edge options

    Release lines¶

    Latest development builds targeting Debian 13 (Trixie) and Armbian 24.8. Rolling updates, not production-ready.

    Stable long-term support on Debian 12 (Bookworm). Recommended for most users.

    Previous long-term support on Debian 11 (Bullseye). Maintenance-only.
